Linguagem Portugol

O Portugol é uma linguagem de programação fictícia utilizada no ensino de lógica de programação. Ela não é uma linguagem de programação real, mas sim uma ferramenta pedagógica criada para facilitar o aprendizado dos conceitos fundamentais da programação de computadores. O termo "Portugol" é uma junção das palavras "Português" e "Algoritmo". As características do Portugol são projetadas para serem próximas da linguagem natural em português, tornando-a mais acessível para iniciantes que estão aprendendo a programar. A ideia é que, ao usar uma linguagem mais próxima do idioma do aprendiz, o processo de compreensão e aplicação dos conceitos básicos de programação seja mais fácil.

    Vantagens de começar no Portugol

  1. Comandos Simples: Estruturas de controle de fluxo de fácil compreensão, como "se...então...senão" e "enquanto...faça".
  2. Variáveis: Introdução ao conceito de variáveis e atribuição de valores.
  3. Entrada e Saída de Dados: Instruções para receber dados do usuário (entrada) e exibir resultados (saída).
  4. Operadores: Uso de operadores aritméticos, lógicos e relacionais.
  5. Procedimentos e Funções Simples: A ideia de dividir um programa em partes menores (procedimentos ou funções) para facilitar a compreensão.
  6. Listas e Estruturas Simples: Introdução a estruturas de dados básicas.
Tipo de Variavel
  • Inteiro (inteiro): Armazena números inteiros, sem parte decimal.
    Exemplo:
    inteiro idade
    idade = 25
  • Real (real): Armazena números com parte decimal.
    Exemplo:
    real altura
    altura = 1.75
  • Caractere (caractere): Armazena um único caractere.
    Exemplo:
    caractere letra
    letra = 'A'
  • Lógico (logico): Armazena sequências de caracteres (strings).
    Exemplo:
    logico aprovado
    aprovado = verdadeiro
  • Texto (literal): Armazena um único caractere.
    Exemplo:
    literal nome
    nome = "João"
Operadores Aritméticos
  • Soma (+): Adiciona dois valores.
    Exemplo:
    resultado <- 5 + 3
  • Subtração (-): Subtrai o segundo valor do primeiro.
    Exemplo:
    resultado <- 7 - 2
  • Multiplicação (*): Multiplica dois valores.
    Exemplo:
    resultado <- 4 * 6
  • Divisão (/): Divide o primeiro valor pelo segundo.
    Exemplo:
    resultado <- 10 / 2
  • Resto (%): Retorna o resto da divisão do primeiro valor pelo segundo.
    Exemplo:
    resultado <- 15 % 4
  • Potenciação (^): Eleva o primeiro valor à potência do segundo.
    Exemplo:
    resultado <- 2 ^ 3
Operadores Relacionais
  • Igual a (==): Verifica se dois valores são iguais.
    Exemplo:
    resultado <- (5==5)
  • Diferente de (!=): Verifica se dois valores são diferentes.
    Exemplo:
    resultado <- (10 !=7)
  • Maior que (>): Verifica se o primeiro valor é maior que o segundo.
    Exemplo:
    resultado <- (8> 3)
  • Menor que (<): Verifica se o primeiro valor é menor que o segundo.
    Exemplo:
    resultado <- (4 < 9)
  • Maior ou igual a (>=): Verifica se o primeiro valor é maior ou igual ao segundo.
    Exemplo:
    resultado <- (6>= 6)
  • Menor ou igual a (<=): Verifica se o primeiro valor é menor ou igual ao segundo.
    Exemplo:
    resultado <- (3 <=5)
Entrada de Dados e Saida de Dados

A entrada e saída de dados são partes essenciais de qualquer programa, permitindo a interação com o usuário e a manipulação de informações. No Portugol, você pode usar os comandos leia para entrada de dados e escreva para saída de dados. Aqui está um exemplo combinando ambos:

// Entrada de dados: solicita ao usuário que insira informações
escreva("Digite seu nome: ")
leia(nome)

escreva("Digite sua idade: ")
leia(idade)

escreva("Digite sua altura em metros: ")
leia(altura)

// Saída de dados: exibe as informações fornecidas pelo usuário
escreva("Seu nome é ", nome, ".")
escreva("Você tem ", idade, " anos.")
escreva("Sua altura é ", altura, " metros.")
Estruturas de repetição

Enquanto

A estrutura enquanto executa uma sequência de comandos enquanto uma condição lógica especificada for verdadeira. O bloco de comandos continuará a ser executado repetidamente até que a condição torne-se falsa.
enquanto x < 10 faca
escreva(x)
x <- x + 1
fim-enquanto

Para

A estrutura para é utilizada quando você sabe antecipadamente quantas vezes deseja que um bloco de comandos seja repetido. Você especifica uma variável de controle, um valor inicial, um valor limite e, opcionalmente, um incremento.
para i de 1 ate 5 faca
escreva(i)
fim-para

Repita-Até

A estrutura repita-até executa uma sequência de comandos até que uma condição lógica especificada seja verdadeira. O bloco de comandos será repetido pelo menos uma vez, independentemente da condição inicial.
repita
escreva("Digite a senha: ")
leia(senha)
ate senha == "1234"
Estruturas de seleção

Se...Então...Senão

A estrutura se...então...senão é usada para executar diferentes blocos de código com base em uma condição lógica.
se idade >= 18 entao
escreva("Você é maior de idade.")
senao
escreva("Você é menor de idade.")
fimse

Escolha...Caso

A estrutura escolha...caso é usada quando há várias condições possíveis e você deseja executar diferentes blocos de código com base no valor de uma expressão.
escolha diaSemana
caso 1
escreva("Domingo")
caso 2
escreva("Segunda-feira")
caso 3
escreva("Terça-feira")
caso 4
escreva("Quarta-feira")
caso 5
escreva("Quinta-feira")
caso 6
escreva("Sexta-feira")
caso 7
escreva("Sábado")
outro
escreva("Número inválido de dia.")
fimescolha
Estruturas do codigo

Vamos criar um simples código em Portugol que calcula a média de um aluno e determina se ele foi aprovado ou reprovado com base em um critério de nota mínima. Vamos assumir que a nota mínima para aprovação seja 6.0.

algoritmo CalculaAprovacao
var
nota1, nota2, media: real

inicio
// Solicita ao usuário que digite suas notas
escreva("Digite a nota da primeira prova: ")
leia(nota1)

escreva("Digite a nota da segunda prova: ")
leia(nota2)

// Calcula a média das notas
media <- (nota1 + nota2) / 2

// Verifica se a média é maior ou igual a 6.0 para aprovação
se media >= 6.0 entao
escreva("Parabéns! Você foi aprovado com média ", media)

senao
escreva("Infelizmente, você foi reprovado com média ", media)
fimse
fimalgoritmo
